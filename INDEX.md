# The 3Deer Portfolio

Showcasing modern Android 3D graphics, an open-source rendering engine, and powerful model viewer applications.

---

## android-model-engine

A standalone 3D rendering engine library for Android implementing a Scene Graph architecture and OpenGL ES 2.0/3.0 support.

## android-model-viewer

A feature-rich application to load, inspect, and visualize 3D models (OBJ, STL, DAE, GLTF, FBX) using our custom 3D engine.

## android-game-isomodel

A 3D game designed to train your brain neurons. Keep young.
